# Bharat Sahayak 🇮🇳

## 🚀 Overview
Bharat Sahayak is an AI-powered assistant designed to help users get information and support through a simple chat interface.

## ✨ Features
- AI Chat Assistant
- Fast Responses
- User-Friendly Interface
- REST API Backend

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- OpenAI API
- MySQL

## ⚙️ Run Locally

```bash
git clone <https://github.com/aman12-droid/bharat-sahayak>
cd project
npm install
npm start
